# Global-Solution---Python

# Resumo do Projeto: O programa é um sistema interativo que educa os usuários sobre a Síndrome de Kessler, um cenário onde o acúmulo de detritos espaciais pode causar uma reação em cadeia de colisões. Ele oferece conteúdo informativo, um quiz para testar o conhecimento dos usuários e um protótipo de gerenciamento de detritos espaciais, permitindo que os usuários detectem, capturem, armazenem e enviem detritos para reciclagem. O objetivo é aumentar a conscientização sobre os riscos da Síndrome de Kessler e promover a sustentabilidade no espaço.
